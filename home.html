  • FRP

    Factory Reset Protection ( FRP )

    ലോലിപോപ്പ് 5.1 ഉം അതിലും ഉയർന്നതുമായ Android ഉപകരണങ്ങളിലെ സുരക്ഷാ സവിശേഷതയാണ് ഫാക്ടറി റീസെറ്റ് പ്രൊട്ടക്ഷൻ (FRP). ആൻഡ്രോയ്ഡ് ഫോണിൽ ഗൂഗിൾ അക്കൗണ്ട് സജ്ജമാക്കുമ്പോൾ FRP ആക്ടീവ് ആകും.ഈ സുരക്ഷാ സവിശേഷത ആക്ടീവാണെങ്കിൽ ഫാക്ടറി റീസെറ്റ് ചെയ്ത ശേഷം ഫോണിൽ ലോഗിൻ ചെയ്തിരുന്ന ഗൂഗിൾ അക്കൗണ്ട് നൽകിയാൽ മാത്രമേ ഫോൺ ഉപയോഗിക്കാനാവൂ.


  • Safe Mode

    Safe Mode

    Safe Mode ഓണായിരിക്കുന്ന ആൻഡ്രോയ്ഡ് ഫോണിൽ നോർമൽ മോഡിലുള്ള ഫോണിനെക്കാൾ കൂടുതൽ FAST ആയിരിക്കും.ഫോണിൻ്റെ Slow,ഹീറ്റിംഗ് പ്രശ്നം എന്നിവ ഫോണിൽ ഇൻസ്റ്റാൾ ചെയ്തിരിക്കുന്ന ആപ്പുകൾ കാരണമാണോ എന്നറിയാൻ Safe Mode ൽ ഫോൺ ഉപയോഗിച്ചാൽ മനസിലാക്കാം.Safe Mode ഓണായിരിക്കുമ്പോൾ റാം ഉപയോഗിക്കുന്ന ഇൻസ്റ്റാൾ ചെയ്തിരിക്കുന്ന ആപ്ലിക്കേഷനുകൾ പ്രവർത്തിക്കില്ല.


  • Radiation/Sar

    SAR Value

    മൊബൈൽ ഫോണിലെ റേഡിയേഷൻ്റെ തീവ്രത അളക്കുന്ന അളവുകോലാണ് SAR (Specific Absorption Rate).നാം മൊബൈൽ ഫോൺ ഉപയോഗിക്കുമ്പോൾ ശരീരം ആഗിരണം ചെയ്യുന്ന ഉയർന്ന ഫ്രീക്വൻസിയിലുള്ള ഊർജത്തിന്റെ അളവാണ് SAR എന്നതുകൊണ്ട് സൂചിപ്പിക്കുന്നത്.SAR അളക്കുന്നത് വാട്ട്/കിലോഗ്രാമിലാണ്.നിയമപ്രകാരം 1.6 വാട്ട്/കിലോഗ്രാമാണ് ഒരു മൊബൈൽ ഫോണിൽ നിന്ന് പുറപ്പെടുവിക്കാവുന്ന പരമാവധി SAR Value.മൊബൈൽ ഫോണിൽ *#07# ഡയൽ ചെയ്താൽ SAR Value അറിയാൻ കഴിയും.

    MEDTALK - Radiation and Life - Dr. K. Mohanan |... Monsoon Media / Jan 22, 2017


  • imei

    IMEI

    International Mobile Equipment Identity എന്നതിനെ ചുരുക്കി പറയുന്നതാണ് IMEI.ഓരോ മൊബൈലിലും വ്യത്യസ്തമായിരിക്കും IMEI നമ്പർ.ഫോൺ നഷ്ടപ്പെട്ടാൽ കണ്ടെത്തുന്നതിന് IMEI നമ്പർ ആവശ്യമാണ്.*#06# ഡയൽ ചെയ്താൽ IMEI നമ്പർ അറിയാൻ കഴിയും. മൊബൈൽ ഫോണിൽ കമ്പനിക്ക് കിട്ടുന്ന അംഗീകാരം,കമ്പനിയുടെ മോഡൽ നിർമ്മാണ നമ്പർ,ഒരു കമ്പനിയുടെ വിവിധ മോഡലുകളുടെ പ്രൊഡക്ഷൻ എണ്ണം എന്നിവയെല്ലാം IMEI നമ്പറിൽ അടങ്ങിയിരിക്കുന്നു.
